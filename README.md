# Dev Toolkit

A collection of precision developer utilities — no noise, no clutter, just what you need. Runs entirely in the browser with zero server-side processing.

**[Live Demo → dev-toolkit-bice.vercel.app](https://dev-toolkit-bice.vercel.app/)**

---

## Tools

| # | Tool | What it does |
|---|------|-------------|
| 01 | **Cron Builder** | Build cron expressions visually, preview next run times, common templates |
| 02 | **JWT Debugger** | Decode tokens, highlight security issues, verify claims, generate new tokens |
| 03 | **SQL Formatter** | Beautify SQL with dialect support, visualize execution plans, get optimization tips |
| 04 | **CSV / JSON Transform** | Filter, map, group, sort, join — live preview table and one-click export |
| 05 | **Image Optimizer** | Client-side compression, WebP/AVIF conversion, responsive srcset generation |
| 06 | **JSON Schema Form** | Paste any JSON Schema → get a live functional form + exportable react-hook-form code |
| 07 | **Regex Tester** | Real-time testing with token-by-token visual breakdown and a pattern library |
| 08 | **Git Command Builder** | Compose complex git commands visually — rebase, cherry-pick, stash, reset, bisect |
| 09 | **Favicon Generator** | Upload one image, get all 12 required sizes + ready-to-paste HTML `<head>` tags |

---

## Stack

- **React 19** + **TypeScript**
- **Tailwind CSS 4** (Vite plugin, zero config)
- **Vite 7** with `vite-plugin-singlefile` — ships as a single self-contained HTML file
- All processing is **100% client-side** — nothing leaves your browser

---

## Run Locally

```bash
git clone https://github.com/your-username/dev-toolkit.git
cd dev-toolkit
npm install
npm run dev
```

Build for production:

```bash
npm run build   # outputs a single dist/index.html
```
